## 🚀 Installation  

1. **Clone the Repository**:  
   ```bash  
   git clone https://github.com/Dhritiman1511/myPortfolio.git 
   cd myPortfolio  
   ```  

2. **Install Dependencies**:  
   ```bash  
   npm install  
   ```  

3. **Run the Project**:  
   ```bash  
   npm run dev 
   ```  

4. **Open in Browser**:  
   Visit `http://localhost:3000` to see the website in action.
